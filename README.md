# HFile

HFile is a lightweight, terminal-based transfer tool in C built around a minimal, explicit protocol.

## how to build

See `./build.sh -h` for build usage.

## how to test

Run `./test.sh` for the full test suite.
Run `./test.sh transfer` for the transfer suite.
Run `./test.sh -v test.test_transfer` if you want raw `unittest` arguments.
