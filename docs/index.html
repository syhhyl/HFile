HFile

A lightweight LAN file transfer tool written in C. One server process exposes a native CLI protocol, an HTTP API, and a browser Web UI on the same port.

GitHub Releases README

Install on macOS and Linux

Installs the latest supported release from GitHub Releases and verifies it with checksums.

curl -fsSL https://syhhyl.github.io/HFile/install.sh | bash

Install on Windows

Use PowerShell to download and run the latest installer script.

irm https://syhhyl.github.io/HFile/install.ps1 | iex

Published targets

  • macOS arm64
  • Linux amd64
  • Windows amd64

Quick start

hf -d ./received
hf -c ./hello.txt
hf -m "hello from CLI"

HFile serves CLI transfer, HTTP API, and Web UI on one port.
